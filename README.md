# hustler
Side Hustle Planner - Stop Guessing. Start Earning.

This project helps you find and plan side hustles based on your skills, goals, and time availability.

## Features
- AI-powered hustle generator
- Personalized roadmaps
- Learning hub with curated resources
- Earnings tracker

## Tech Stack
- React + Vite
- Recharts for data visualization
- Modern CSS (Syne, Outfit, Space Mono fonts)

## Development
```bash
npm install
npm run dev
```
